# Professional JavaScript for Web Developers

## Javascript 实现
一个完整的 JavaScript 实现应该由下列三个不同部分组成
- 核心（ ECMAScript ）
    - ECMAScript 与 Web 浏览器没有依赖关系。这门语言本身并不包含输入和输出定义。
    - ECMA-262 规定了这门语言的下列组成部分：
        - 语法
        - 类型
        - 语句
        - 关键字
        - 保留字
        - 操作符
        - 对象
- 文档对象模型（ DOM ）
    - 是
- 浏览器对象模型（ BOM ）
